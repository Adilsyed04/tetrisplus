# TetrisPlus Proof of Concept
This repository contains a small proof-of-concept for TetrisPlus, an extensible Tetris clone that is written in Java using Swing. 

This code is modeled from Tetris and Java Swing game tutorials. It's meant to show how the concept works and how it'll look like in the beginning stages.

Tested on:
Windows 11 (64-bit),
Java Development Kit (JDK) 21,
VS Code

Steps:
1. Clone Repository into VS Code
2. Open a terminal prompt in the directory
3. Compile with: javac *.java
4. Run with: java TetrisPlusMain
5. Controls:
Left Arrow: Move Piece Left,
Right Arrow: Move Piece Right,
Up Arrow: Rotate Piece,
Down Arrow: Soft Drop (Moves Down 1 Step)
